# 📅 주간 업무 회고 (2월 1주차)

## 1. 이번 주 요약 (Executive Summary)

이번 주는 **기존 법무법인 홈페이지의 유지보수**와 **신규 CRM 시스템의 인프라 구축 및 핵심 기능 구현**에 집중했습니다.

특히 CRM 프로젝트의 경우, **인증(OAuth)부터 배포(Railway), DB 설계**까지 백엔드 전반의 기초 공사를 완료하였으며, 시스템 아키텍처를 구체화하여 데이터 흐름을 정립한 것이 주요 성과입니다. 또한, 개발 효율성을 높이기 위해 AI 툴(Cursor) 업그레이드를 요청하여 업무 환경 개선을 시도했습니다.

---

## 2. 프로젝트별 상세 성과

### 🅰️ CRM 개발 (BS_CRM)

가장 많은 리소스가 투입된 프로젝트로, 로그인 프로세스 구현과 서버/DB 인프라 구축이 완료되었습니다.

- **인증 및 보안 (Authentication & Security)**
    
    - **Google OAuth 연동:** 웹/모바일 이중 플랫폼 로그인 프로세스 구현.
        
    - **보안 강화:** 회사 도메인(`@law-win.co.kr`) 계정만 로그인 가능하도록 제한 로직 적용 및 토큰 검증 API 구현.
        
    - **계정 연동:** 구글 계정 연동 기반 모바일 연락처 저장 기능 개발.
        
- **인프라 및 배포 (Infrastructure & DevOps)**
    
    - **서버 구축:** Railway를 이용한 Node.js 22 (Docker) 프로덕션 배포 환경 구축 완료.
        
    - **환경 분리:** 운영(Production) 및 개발(Dev) 환경에 따른 CORS, API URL 자동 분기 처리.
        
    - **안정성 확보:** Service Account 예외 처리 로직을 추가하여 배포 환경 크래시 방지 및 외부 접근 권한(Binding) 설정 최적화.
        
- **데이터베이스 (Database)**
    
    - **스키마 설계:** 초기 DB 구조 설계 및 테이블 명칭 리팩토링 (`party` → `party_detail`) 및 참조 로직 수정.
        
    - **마이그레이션:** 웹·앱 공통 SQL 연동 및 배포를 위한 마이그레이션, 로컬 DB 덤프 및 CLI 접속 테스트 완료.
        
- **시스템 아키텍처 (Architect수 (law-win)

사용자 경험 개선 및 데이터 분석 환경 마련에 중점을 두었습니다.

- **다국어 지원:** CN(중국어), TH(태국어), EN(영어) 전환 UI/기능 적용 및 미적용 문구 수정 완료.
    
- **사용자 분석:**
    
    - 천안 지사 페이지에 **Microsoft Clarity** 도입 (CRM 사용성 및 행동 패턴 분석).
        
    - Google Analytics 연동.
        
- **SEO 최적화:** 메인 홈 화면 구조화 데이터(`ItemList`)에 전국 지사 명칭 정보 추가하여 검색 노출 효율 증대.

## 3. 기술적 이슈 및 해결 (Troubleshooting)

|**이슈 항목**|**내용 및 해결 방안**|
|---|---|
|**배포 환경 충돌**|**문제:** Service Account 관련 설정 미비로 배포 환경에서 크래시 발생<br><br>  <br><br>**해결:** 예외 처리 로직 추가 및 Google Admin Directory API 선택적 활성화 적용|
|**DB 연결성**|**문제:** 서버 환경에서 MySQL 접속 불안정<br><br>  <br><br>**해결:** 포트(Port) 설정을 명시적으로 추가하여 접속 안정성 확보|
|**보안 취약점**|**문제:** 외부 계정 접근 가능성<br><br>  <br><br>**해결:** OAuth 로그인 시 도메인 화이트리스트(`@law-win.co.kr`) 필터링 로직 강제 적용|
